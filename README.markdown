# GlobalAccess IP Scanner

---

# 🇬🇧 English

## Overview

GlobalAccess IP Scanner is a professional multi-threaded IP range scanner and connectivity analyzer built with Python and PyQt6.

The application provides advanced network diagnostics including:

- IP Range & CIDR Scanning
- ICMP Ping Testing
- TCP Port Validation (80 / 443)
- TLS Handshake Verification
- HTTPS Connectivity Testing
- DPI / Filtering Detection
- Traceroute Analysis
- Latency & Jitter Measurement
- Packet Loss Analysis
- Health Scoring System
- JSON Export
- CSV Export
- Multi-language Interface (English, Persian, Chinese)
- Dark & Light Themes

## Features

### Network Analysis
- Fast multi-threaded scanning
- CIDR support

- Individual IP scanning
- Reachability checks

### Requirements

- Python 3.10+
- Windows / Linux / macOS

```bash
pip install PyQt6
```


### IP Range

